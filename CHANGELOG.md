# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2024-11-29

### Added
- Initial release with energy consumption estimation for household appliances
- Query EnergyStar API for appliance energy consumption data
- Support for multiple appliance types
- Estimator class with median and mean aggregation strategies
- Case-insensitive brand and model queries
